# Contacts App

An Android Contacts Application built using Clean Architecture, MVVM, and Jetpack Compose.

#  Features
- Add Contact
- Update Contact
- Delete Contact
- Search Contacts
- Image Picker (stored in internal storage)

# Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- Hilt (Dependency Injection)
- Coroutines + Flow

# Architecture
Clean Architecture:
- data
- domain
- presentation
